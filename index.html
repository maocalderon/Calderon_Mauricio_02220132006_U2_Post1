Mis Proyectos Académicos

Proyecto 01: Análisis de Datos

En este proyecto utilicé herramientas de recolección de datos para analizar el rendimiento estudiantil.

Fue una experiencia enriquecedora para entender la importancia de la información estructurada.

Proyecto 02: Maquetación Web

Desarrollé la estructura base de un blog técnico utilizando etiquetas semánticas de HTML5.

El enfoque principal fue mantener un código limpio y fácil de mantener para futuros cambios.

Proyecto 03: Aplicación Local

Documentación de un pequeño sistema de gestión de inventarios para una tienda local en Cúcuta.

Este proyecto me permitió practicar la lógica de programación y la interacción con usuarios.
